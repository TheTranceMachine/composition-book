# Composition book

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```

For a detailed explanation on how things work, check out the [guide](http://vuejs-templates.github.io/webpack/) and [docs for vue-loader](http://vuejs.github.io/vue-loader).

## Description

This app should help anyone to write a short story or an essay. 
Every story should be part of a bigger world. 
Every world can have a number of characters living in the world.
After all the above is created, the author should be able to link/refer to the world and characters in the story at any point.
